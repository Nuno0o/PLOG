Javaploy 

For latest Information visit:
-----------------------------
http://javaploy.sourceforge.net


Contents of the base directory:
-------------------------------
build.xml 
  - the configuration file for ANT
src 
  - contains java source files
doc 
  - contains documentation for the project
doc/javadoc
  - contains javadoc documentation (created by ANT)
bin
  - contains java byte code (created by ANT)
dist
  - contains jar files, for the applet and/or applications
copyright.txt
  - contains copyright information
install.txt
  - contains installation information
todo.txt
  - tasks thinking of doing in the future
